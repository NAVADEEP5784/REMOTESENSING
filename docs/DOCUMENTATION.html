01 EXECUTIVE SUMMARY

The Satellite Intelligence Platform is a full-stack, AI-powered remote sensing and image analysis web application. It provides land cover classification (10 EuroSAT classes via ResNet18), object detection (YOLOv8), AI chatbot explanations (OpenAI GPT), and an admin dashboard with authentication.

10 INSTALLATION & DEPLOYMENT GUIDE

REQUIREMENTVERSION

Python3.10+
OpenAI API KeyFor chatbot

cd ai-service
pip install -r requirements.txt
uvicorn main:app --reload --host 0.0.0.0 --port 8000

STEP 2 — BACKEND

cd backend
npm install
# Set .env: PORT, AI_SERVICE_URL, OPENAI_API_KEY, ADMIN_EMAIL, ADMIN_PASSWORD, JWT_SECRET
npm run dev

STEP 3 — FRONTEND

cd frontend
npm install
npm start

Start order: 1) AI (8000), 2) Backend (5000), 3) Frontend (5173)

12 DATA FLOW ARCHITECTURE

ANALYSIS FLOW:

User selects image → POST /api/analyze (multipart) → Express receives → Multer saves to uploads/ → Proxy to FastAPI /analyze → Classifier + Detector run → Response (classification, detections, base64) → Frontend setAnalysis → Navigate to /results → Charts + insights render.

CHAT FLOW:

User sends message → POST /api/chat (message, classification, detections) → Express → OpenAI API → Reply returned → Display in chat.

HISTORY FLOW:

After successful analysis → addToHistory(analysis, imgDataUrl) → localStorage → History page reads and displays grid.
